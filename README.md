### Instalace

#### Instalace závislostí

Před prvním spuštěním serveru je zapotřebí stáhnout všechny balíčky

```
npm i
```

#### Spuštění aplikace

Jako vývojář

```
npm run dev
```

Nebo aplikaci prvě postavíme

```
npm run build
```

```
npm run start
```
